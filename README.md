一款基于人工智能的游戏屏幕翻译工具，支持截图识别、文本翻译和主题自定义，专为游戏玩家和外语学习者设计。
✨ 功能特点
📸 智能截图翻译
全屏截图：支持多显示器环境，任意区域截图

OCR文字识别：集成Tesseract OCR引擎，精准识别游戏文本

一键翻译：按F10快速截图并翻译

🤖 AI智能翻译
DeepSeek API：使用先进的AI模型进行翻译

上下文理解：保持语义连贯，理解游戏特定术语

多语言支持：主要支持英文到中文翻译，可扩展其他语言

🎨 个性化界面
主题切换：四种精心设计的主题（白色、黑色、卡其色、暗灰色）

字体自定义：可调整字体类型和大小，提高可读性

窗口定制：自由调整窗口大小和位置，保持置顶显示

⚡ 便捷操作
热键支持：F10截图、F9显示/隐藏窗口、Esc退出

拖拽调整：自定义标题栏拖拽移动，右下角手柄调整大小

最小化替代：隐藏窗口到后台，随时恢复

📦 安装指南
系统要求
Python 3.8 或更高版本

Windows 7+/macOS 10.13+/Linux (Ubuntu 18.04+)

至少4GB RAM
